珠峰培训最新课程体系梳理（试改）

正式课阶段周期
2：两周基础免费招生课
7：7周的JS/ES6/设计模式/移动端/AJAX
5：三周vue/node + 两周react
2：两周项目实战，周六日找企业实战老师过来讲课，vue+react实战各一个（期间简历、毕设、就业指导都完成）

共计：16周

正式课第一周

  • Git基础

目标：学生安装了git / node / markdown / vs / ws 等常用的软件和工具，初步可以基于git，上传下载讲义或者其它内容，掌握一些常用的dos / linux / npm 操作命令（逐步养成命名操作的模式和习惯）

  • 闭包和堆栈内存释放问题
    • 变量提升
    • 作用域和作用域链
    • 堆栈内存及内存释放：闭包（闭包的作用）
    • this基础
    • 案例练习：面试题+实战项目案例
    • ES6 中 let/const 和 ES5 中 var 的区别

目标：熟练掌握闭包在真实项目中的作用（面试必问），能够轻松驾驭关于作用域类的面试题，掌握ES6新语法的特点，初步了解到ES6语法的严谨性，个人建议本阶段就是大量做题（在做题中把arguments的映射以及逻辑或和逻辑与等信息就讲了）

  • 面向对象 (OOP)
    • 单例设计模式及实战用途（初步接触模块化开发）
    • 工厂模式
    • 构造函数模式
    • instanceof
    • hasOwnProperty
    • in
    • 原型及原型链
    • 基于内置内类的原型扩展方法，实现链式写法，以及里面涉及的this问题
    • 原型设计模式中的一些细节，例如：原型重定向导致的constructor重写问题

目标：有面向对象编程的思想和意识，这节课程是为了后期学习更深入的知识，以及实现组件封装打基础，比较的抽象难理解，但是也非常的重要（建议也是做更多的题，来强化大家的理解，让学生们能够自己画出原型图）

正式课第二周

  • 原型和this深入
    • 函数的多种角色（阿里面试题）
    • call / apply / bind 的区别及实战项目中的应用
    • fn1.call.call.call(fn2)
    • 求数组中的最大值
    • 获取平均数：把类数组转换为数组
    • ES6中的新语法：解构赋值、...的作用、形参默认值、形参直接解构、箭头函数等（对比着ES5来讲）
    • this汇总

目标：熟练掌握this的指向，了解 call / apply / bind 在项目中的用途，掌握更多的ES6新语法；函数的多种角色是重点，为后期讲JQ原理做铺垫。

  • 商城排序
    • 简单的AJAX四步操作
    • JSON
    • 基于数组的SORT进行排序（SORT原理）
    • 数据绑定：ES6中的模板字符串（可以讲一下其它的方法，createDocumentFragement）
    • DOM的回流重绘机制
    • DOM的映射机制

目标：在案例中不断强化this及基础知识的应用，慢慢培养大家的编程思想，关于DOM的回流重绘机制这个非常的重要，是前端优化重要的操作方式

  • 正则
    • 常用的元字符修饰符，重点是：[] () ? …
    • 常用正则表达式编写
    • 创建正则的两种方式及区别
    • 正则捕获：解决懒惰性和贪婪性（要分析懒惰性的原理）
    • 更多的正则捕获方法：test / exec / match / replace … 要对比方法之间的区别等
    • 案例练习：URL问号参数截取、时间字符串格式化、获取出现最多的字母及次数、千分符…
    • 表单验证（可以放到后面项目实战中讲也可以）
    • 正负向预查

目标：正则是珠峰的一个特色课程，没有那么重要，但是也要让学生能够写出一些常用的正则，能够分析别人写好的正则等，能够基于正则去优化封装项目中一些常用的方法…

正式课第三周

  • 盒子模型
    • css盒子模型和css3盒子模型
    • 盒子模型常用的属性，以及获取值的特点
    • get/set/setGroupCss方法的封装（可以不解决兼容问题）
    • 盒子偏移量（offset方法封装）
    • 滚动条卷去的高度
    • 回到顶部 (初步接触定时器)
    • 跑马灯 （选）
    • 京东楼层导航 （选）
    • …

目标：掌握13个属性的意义，能够基于13个属性完成一些小的需求，封装的这几个方法没啥实际意义，一方面为后面的JQ做准备，一方面培养大家一些封装处理的思想

  • jQuery
    • jQuery下载（基于npm 或者 yarn 下载指定版本的）
    • JQ源码解读
      • 整体构架：$()是创建JQ的一个实例
      • 方法存在两部分：原型和对象上，基于extend扩展方法
      • 选择器传参类型不同，结果不同
      • JQ对象和原生JS对象相互转换
      • JQ放弃对于$的使用权
      • …
    • JQ常用的方法（对照API讲解），重点讲解的方法（最好对照源码讲解）
      • each
      • filter / find / children
      • …
    • 基于JQ做点案例（例如：重写选项卡，也可以把锋利JQ中的一些案例搬过来做，根据时间自由把控）

目标：现在社会上JQ的使用越来越少，太多的学习JQ已经失去实际的意义，但是我们可以深入解读JQ的源码，培养大家慢慢看懂JQ源码，通过学习源码提高我们的编程思想及封装技巧等；而JQ的用法，后面多做几个案例就差不多了。

  • 瀑布流案例
  • 延迟加载案例
  • JQ中的一些方法（或者一些常用方法）自己去封装 [ 选 ]

目标：这个阶段的案例用JQ做也可以，用ES6/ES5做也可以，这个无所谓，重点是培养大家编程能力（初步慢慢的养成改BUG的能力）

三周汇总专题

  • JS中的继承（ES6中的class类）
  • JS中的数据类型检测
  • JS中的THIS指向问题
  • ES6部分新特性等

这部分可以通过做阶段练习（大量做面试题）来查漏补缺，来进行知识体系的梳理等；前三周是一个阶段，讲解了很多JS核心操作原理，这些都非常的重要！

第四周

  • 定时器和异步编程
    • setTimeout
    • setInterval
    • 异步编程的原理
    • 初始Promise
  • 封装动画库
    • 简单匀速和非匀速动画的实现
    • 封装动画库
    • 回调函数机制：重写forEach / replace 等
    • JQ中的动画及动画库
  • CSS3中的动画
  • window.requestAnimationFrame [ 选 ]

目标：动画的一些梳理，项目中动画很少，大部分也都是基于CSS3来完成，所以重点不是动画。封装动画库是培养大家的编程思想，让大家对于ES6有具体的认识，回调函数很重要，异步编程更重要，这些是我们需要重点去讲解的

  • 实战案例：轮播图
    • JQ版：渐隐渐现
    • JS版：左右运动（无缝衔接）
    • 封装成为轮播图插件（基于面向对象封装）

目标：本课程的难点在于基于面向对象的方式封装插件，这一块是难点也是重点，不需要所有人学会，重点培养尖子生

第五周

  • 事件及事件对象
    • 常用的PC和键盘事件
    • 事件对象及兼容
    • 事件传播机制
    • 事件委托
    • 移动端事件
  • 案例练习
    • 京东放大镜
    • 鼠标跟随
    • MENU菜单
    • 多级菜单
    • 拖拽及弹性势能动画

目标：重点是讲解事件在真实项目中的应用，所以实战案例为主要讲解目标

  • DOM2事件及事件池机制
  • 发布订阅设计模式(ES6)
  • 基于发布订阅封装DIALOG插件(ES6)

目标：设计模式讲解，让大家了解事件池运行机制，培养大家整体的编程和封装能力，这部分比较难，允许部分学员不会

目标：第六周为实战周，把前面学的知识统一进行应用，让学生以后想做什么都可以做（不基于框架），在实战案例中，把SEO、CSS3、H5等基础知识贯穿讲解，充分应用ES6中的一些常用语法…
